# Ansible Provider

Wraps [Ansible](https://www.ansible.com/) to run playbooks and ad-hoc tasks as part
of your hams environment setup.

**Platforms:** macOS, Linux

## Usage

```bash
hams ansible run <urn>
hams ansible list
```

## Hamsfile Example

```yaml
- urn: "urn:hams:ansible:setup-dev-env"
  step: Configure development environment
  playbook: playbooks/dev-env.yml
  check: test -f ~/.dev-env-configured
```
