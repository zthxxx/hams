# pnpm Provider

Wraps [pnpm](https://pnpm.io/) to manage globally installed Node.js packages.

**Platforms:** macOS, Linux

## Usage

```bash
hams pnpm add <package>
hams pnpm remove <package>
hams pnpm list
```

## Hamsfile Example

```yaml
- app: serve
  intro: Static file serving and directory listing.
- app: tsx
  intro: TypeScript execute for Node.js.
```
