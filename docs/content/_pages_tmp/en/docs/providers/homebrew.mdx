# Homebrew Provider

Wraps [Homebrew](https://brew.sh/) (`brew`) to manage formulae and casks on macOS and Linux.

**Platforms:** macOS, Linux

## Usage

```bash
hams brew install <formula>
hams brew install --cask <cask>
hams brew remove <formula>
hams brew list
```

## Hamsfile Example

```yaml
- app: ripgrep
  intro: Fast recursive grep alternative.
- app: bat
  tags: [cli]
  intro: A cat clone with syntax highlighting.
```
