# git clone Provider

<Callout type="info">
  此页面尚未完成翻译，以下为英文原文。欢迎贡献翻译。
</Callout>

Manages repository clones at fixed local paths. Probes by checking whether the target
directory exists and the remote URL matches.

**Platforms:** macOS, Linux

## Usage

```bash
hams git-clone add <remote-url> <local-path>
hams git-clone list
```

## Hamsfile Example

```yaml
- urn: "urn:hams:git-clone:dotfiles"
  step: Clone dotfiles repository
  remote: https://github.com/zthxxx/dotfiles.git
  local_path: ~/Projects/dotfiles
  default_branch: main
```
