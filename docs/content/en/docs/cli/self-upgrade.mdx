# hams self-upgrade

Upgrade the hams binary to the latest release. Downloads the appropriate binary
for your platform from GitHub Releases.

## Usage

```bash
hams self-upgrade [flags]
```

## Examples

```bash
# Upgrade to the latest version
hams self-upgrade

# Check current version before upgrading
hams --version
```
