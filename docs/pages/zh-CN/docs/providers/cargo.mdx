# cargo Provider

<Callout type="info">
  此页面尚未完成翻译，以下为英文原文。欢迎贡献翻译。
</Callout>

Wraps `cargo install` to manage Rust binaries installed from crates.io or git.

**Platforms:** macOS, Linux

## Usage

```bash
hams cargo install <crate>
hams cargo uninstall <crate>
hams cargo list
```

## Hamsfile Example

```yaml
- app: ripgrep
  intro: Fast recursive grep.
- app: fd-find
  intro: Simple and fast alternative to find.
```
