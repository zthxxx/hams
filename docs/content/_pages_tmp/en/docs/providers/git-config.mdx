# git config Provider

Manages Git configuration key-value pairs via `git config`. Probes the current value
with `git config --get` and writes with `git config --global`.

**Platforms:** macOS, Linux

## Usage

```bash
hams git-config set <key> <value>
hams git-config list
```

## Hamsfile Example

```yaml
- urn: "urn:hams:git-config:user-name"
  step: Set git user name
  key: user.name
  value: "Your Name"

- urn: "urn:hams:git-config:rerere"
  step: Enable git rerere
  key: rerere.autoUpdate
  value: "true"
```
