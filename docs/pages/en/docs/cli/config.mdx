# hams config

Read and write hams configuration values. Configuration is stored in
`~/.config/hams/hams.config.yaml` by default.

## Usage

```bash
hams config <subcommand> [flags]
```

## Examples

```bash
# Show the current configuration
hams config show

# Get a specific config value
hams config get store_path

# Set a config value
hams config set store_path ~/Projects/my-hams-store
```
