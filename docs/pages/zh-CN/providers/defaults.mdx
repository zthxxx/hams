# macOS defaults Provider

<Callout type="info">
  此页面尚未完成翻译，以下为英文原文。欢迎贡献翻译。
</Callout>

Wraps `defaults write` to manage macOS system and application preferences. Probes
current values via `defaults read` and compares against the desired state.

**Platforms:** macOS only

## Usage

```bash
hams defaults set <domain> <key> <value>
hams defaults list
```

## Hamsfile Example

```yaml
- urn: "urn:hams:defaults:finder-show-extensions"
  step: Show all file extensions in Finder
  domain: NSGlobalDomain
  key: AppleShowAllExtensions
  value: true

- urn: "urn:hams:defaults:dock-autohide"
  step: Auto-hide the Dock
  domain: com.apple.dock
  key: autohide
  value: true
```
