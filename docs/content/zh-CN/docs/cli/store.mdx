# hams store

<Callout type="info">
  此页面尚未完成翻译，以下为英文原文。欢迎贡献翻译。
</Callout>

Manage the store repository that holds your Hamsfiles and configuration. The store is
a git repository that tracks your declared environment state.

## Usage

```bash
hams store <subcommand> [flags]
```

## Examples

```bash
# Initialize a new store
hams store init

# Clone an existing store from GitHub
hams store clone zthxxx/hams-store

# Show store status
hams store status
```
